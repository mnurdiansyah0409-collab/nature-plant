# Nature Plant

Nature Plant adalah website penjualan tanaman hias, tanaman buah, dan bibit premium. Proyek ini dibangun menggunakan teknologi statis murni (HTML, CSS, JavaScript) dengan fitur pencarian produk, penyaringan kategori, keranjang belanja interaktif, integrasi pemesanan WhatsApp, dan Admin Panel lokal.

## 🛠️ Teknologi
- **HTML5**: Struktur halaman semantik.
- **CSS3**: Sistem desain responsif minimalis, glassmorphism, dan animasi mikro.
- **JavaScript (ES6+)**: Logika katalog dinamis, pengelolaan keranjang, pemesanan WhatsApp, dan manajemen produk (Admin Panel).
- **JSON**: Database produk statis awal (`products.json`).
- **AI-Generated Assets**: Gambar tanaman premium hasil buatan kecerdasan buatan.

## 📂 Struktur Proyek
- `index.html` - Halaman utama aplikasi e-commerce.
- `style.css` - Lembar gaya visual.
- `app.js` - Logika dan interaktivitas website.
- `products.json` - Basis data tanaman awal.
- `assets/` - Aset gambar tanaman berkualitas tinggi.

## 🚀 Cara Penggunaan Lokal
1. Unduh atau klon repositori ini.
2. Buka folder proyek.
3. Klik ganda file `index.html` untuk menjalankannya langsung di browser web pilihan Anda.
